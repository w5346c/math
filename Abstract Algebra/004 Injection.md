Отличный вопрос! Инъекция — это один из трех основных типов функций (наряду с сюръекцией и биекцией). Давайте разберемся просто.

### Что такое инъекция? (Инъективное отображение)

**Простыми словами:** Функция является инъективной, если **разные элементы "входа" всегда переходят в разные элементы "выхода"**.

**Определение:** Функция `f: A → B` называется **инъекцией**, если для любых `x₁` и `x₂` из множества `A` следует:
> Если `f(x₁) = f(x₂)`, то `x₁ = x₂`

Или, что то же самое:
> Если `x₁ ≠ x₂`, то `f(x₁) ≠ f(x₂)`

**Ключевая идея:** Никакие два разных элемента из `A` не "склеиваются" в один элемент в `B`. У каждого элемента `B`, который является значением функции, есть **не более одного** прообраза.

---

### Аналогии

1.  **Очередь в столовой:** Представьте, что каждый человек (элемент из `A`) получает уникальный номерок (элемент из `B`).
    *   **Инъекция:** Все номерки разные. Двум разным людям никогда не достанется один и тот же номерок.
    *   **Не инъекция:** Если два разных человека получат одинаковый номерок, начнется путаница!

2.  **Отпечатки пальцев:** У каждого человека (элемент `A`) есть уникальный отпечаток пальца (значение в `B`).
    *   Это инъективное соответствие, потому что разным людям соответствуют разные отпечатки.

---

### Примеры

#### ✅ Пример 1: Инъекция
*   `A = {Анна, Борис, Виктор}`
*   `B = {1, 2, 3, 4, 5}`
*   Функция `f`: Присвоить людям номерки.
    *   `f(Анна) = 1`
    *   `f(Борис) = 2`
    *   `f(Виктор) = 3`

**Проверка:** Все значения функции (1, 2, 3) — разные. Разным людям соответствуют разные числа. Это **инъекция**.

#### ❌ Пример 2: Не инъекция
*   `A = {Анна, Борис, Виктор}`
*   `B = {1, 2}`
*   Функция `f`: Присвоить людям номерки по полу.
    *   `f(Анна) = 1` (девушка)
    *   `f(Борис) = 2` (юноша)
    *   `f(Виктор) = 2` (юноша)

**Проверка:** `f(Борис) = 2` и `f(Виктор) = 2`. Мы видим, что **разные** элементы `A` (Борис и Виктор) отобразились в **один и тот же** элемент `B` (число 2). Это **НЕ инъекция**.

#### ✅ Пример 3: Математический — Удвоение
*   `A = R` (все действительные числа)
*   `B = R`
*   Функция `f(x) = 2x`

**Проверка:** Предположим, `f(a) = f(b)`. Тогда `2a = 2b`. Делим обе части на 2 и получаем `a = b`. Условие инъективности выполняется. Это **инъекция**.

#### ❌ Пример 4: Математический — Квадрат
*   `A = R` (все действительные числа)
*   `B = R`
*   Функция `f(x) = x²`

**Проверка:** Возьмем `a = 2` и `b = -2`. Они разные (`a ≠ b`), но `f(2) = 4` и `f(-2) = 4`. Получилось, что `f(a) = f(b)`, но `a ≠ b`. Условие инъективности нарушено. Это **НЕ инъекция**.

---

### Как это связано с другими типами функций?

Часто инъекцию рассматривают в связке с двумя другими понятиями. Есть удобная мнемоника:

1.  **Инъекция (Injective) = "Разные входят → разные выходят"**
    *   Проверка: Может ли какой-то элемент `B` получиться из двух разных элементов `A`?

2.  **Сюръекция (Surjective) = "Все элементы B используются"**
    *   Проверка: Для каждого элемента в `B` есть хотя бы один прообраз в `A`?

3.  **Биекция (Bijective) = Инъекция + Сюръекция**
    *   Это "идеальное" взаимно-однозначное соответствие между `A` и `B`.

### Графическая проверка ("Тест горизонтальной линии")

Для функций из `R` в `R` есть простой способ проверить инъективность:
> Если любая **горизонтальная** прямая пересекает график функции **не более чем в одной точке**, то функция инъективна.

*   **`f(x) = 2x`:** Любая горизонтальная линия пересекает график (прямую) ровно один раз. ✅ Инъекция.
*   **`f(x) = x²`:** Горизонтальная линия `y=4` пересекает параболу в двух точках: `x=2` и `x=-2`. ❌ Не инъекция.

### Итог

**Инъекция** гарантирует нам, что информация не "теряется" при переходе от `A` к `B` — по результату `f(x)` мы можем однозначно восстановить, какой именно `x` (какой "вход") был использован, если, конечно, этот результат вообще достижим.
